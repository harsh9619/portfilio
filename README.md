## Harsh Shukla вЂ” React Portfolio

A dark-themed developer portfolio built with React and Vite. This repository contains a single-page portfolio site showcasing projects, experience, skills, and contact information.

Live demo: (add your deployed URL here)

Highlights
- Clean, responsive UI with dark theme and toggle
- Sections for Hero, About, Skills, Experience, Projects, and Contact
- Built with modern React + Vite tooling for fast development

Prerequisites
- Node.js (16+ recommended)
- npm or yarn

Quick start

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

Production build

```bash
npm run build
```

The production output is placed in `dist/`. Serve `dist/` with any static host (Netlify, Vercel, GitHub Pages, etc.).
